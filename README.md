# Project Planner

A modern, high-performance project planning and task management app built natively with **Swift** for **iOS 26**.

Project Planner helps individuals and teams organize projects, manage tasks, track progress, and stay productive with a clean, intuitive interface.

## ✨ Features

* 📋 Create and manage multiple projects
* ✅ Powerful To-Do management
* 📝 Create, edit, and delete tasks
* 🎯 Organize tasks by project
* 📅 Track pending and completed work
* 🔍 Clean and minimal user experience
* ⚡ High-performance native Swift architecture
* 📱 Designed specifically for iOS 26
* 🎨 Modern Apple Human Interface inspired UI
* 💾 Efficient local data management

## 🛠️ Tech Stack

* **Language:** Swift
* **Platform:** iOS 26
* **IDE:** Xcode
* **Architecture:** Native iOS
* **UI:** SwiftUI

## 🚀 Getting Started

### Requirements

* macOS
* Xcode (latest version)
* iOS 26 SDK

### Installation

```bash
git clone https://github.com/sayandeep-coder/Project-Planner.git
cd Project-Planner
```

1. Open the project in **Xcode**.
2. Select an iOS Simulator or a physical device.
3. Press **⌘ + R** to build and run.
